# AI应用开发项目集

基于大语言模型的AI应用开发实践，包含三个核心项目。

## 项目列表

### 1. 简历优化助手
- 通过System Prompt定制AI角色
- 多轮对话，支持持续优化
- 运行：`python hello_ai.py`

### 2. 企业知识库问答（RAG）
- 读取本地文档构建知识库
- AI只根据文档内容回答，不瞎编
- 运行：`python rag.py`

### 3. 代码审查Agent
- 自动调用多个工具完成任务
- 同时进行逻辑分析和Bug检查
- 运行：`python agent.py`

## 技术栈
- Python 3.13
- DeepSeek API
- OpenAI SDK
- ChromaDB
